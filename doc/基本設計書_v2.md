# 基本設計書 Ver2.0

**システム名**：見積書作成ツール\
**版数**：Ver2.0\
**対象**：Microsoft Graph・OneDrive対応版（タブ機能追加前）

------------------------------------------------------------------------

# 1. システム概要

本システムは、業務計画書（PDF）から情報を抽出し、Microsoft Graph
APIを利用してOneDrive／SharePoint上の管理台帳を更新した後、見積書（Excel・PDF）を作成するWindowsデスクトップアプリケーションである。

------------------------------------------------------------------------

# 5. クラス設計

## MainWindow

-   画面制御
-   PDF選択
-   Graph認証開始
-   OneDrive更新開始
-   Excel/PDF出力

## PDFReader

PDFから以下を抽出する。 - 申請書No - 依頼部署 - 件名 - 成果物 -
委託金額 - 納期

## EstimateData

抽出データ保持。

保持項目 - application_no - voucher_no - department - subject -
outputs - amount - due_date - estimate_no - issue_date

## GraphAuth

-   Device Code Flow認証
-   トークンキャッシュ管理

## OneDriveService

-   Graph API接続
-   OneDriveダウンロード
-   OneDriveアップロード
-   ユーザー情報取得

## LedgerWriter

-   対象シート検索
-   C列検索
-   No採番
-   見積番号採番
-   発行日・発行者記入

## ExcelWriter

-   テンプレートコピー
-   セル転記
-   Excel保存
-   PDF出力

## LedgerUpdateWorker

-   バックグラウンドでOneDrive更新

# 10. エラー処理

  対象               処理
  ------------------ ------------------
  PDF未選択          警告
  URL未入力          警告
  Graph認証失敗      エラー
  管理台帳更新失敗   エラー
  423ロック          リトライ後エラー
  412競合            更新中止
  Excel出力失敗      エラー
