# 1. システム概要

## 1.1 システム構成

本システムは、業務計画書（PDF）と管理台帳（Excel）を入力として受け取り、必要な情報を抽出・検索し、見積書フォーマットへ自動転記するWindowsデスクトップアプリケーションである。

```text
┌─────────────────────┐
│      メイン画面      │
└─────────┬───────────┘
          │
          ▼
   PDFReader（PDF解析）
          │
          ▼
 EstimateData（データ保持）
          │
          ├─────────────┐
          ▼             ▼
 LedgerReader      プレビュー表示
 （管理台帳検索）
          │
          ▼
 EstimateData更新
          │
          ▼
 ExcelWriter
 （Excel出力）
          │
          ▼
 見積書.xlsx
```

# 4. クラス設計

## 4.1 MainWindow

### 役割

* メイン画面表示
* ファイル選択
* 読み込み処理
* Excel出力処理
* メッセージ表示

### 主なメソッド

| メソッド           | 内容           |
| -------------- | ------------ |
| setup_ui()     | 画面生成         |
| select_pdf()   | PDF選択        |
| select_excel() | 管理台帳選択       |
| load_pdf()     | PDF解析・管理台帳検索 |
| export_excel() | Excel出力      |

---

## 4.2 PDFReader

### 役割

PDFから必要情報を抽出する。

### 主なメソッド

| メソッド           | 内容      |
| -------------- | ------- |
| read_text()    | PDF全文取得 |
| parse()        | PDF解析   |
| find_next()    | 次行取得    |
| get_amount()   | 委託金額取得  |
| get_due_date() | 納期取得    |
| get_outputs()  | 成果物取得   |

---

## 4.3 LedgerReader

### 役割

管理台帳から見積番号・発行日を取得する。

### 主なメソッド

| メソッド   | 内容     |
| ------ | ------ |
| find() | 管理台帳検索 |

---

## 4.4 ExcelWriter

### 役割

見積書Excelを作成する。

### 主なメソッド

| メソッド                    | 内容       |
| ----------------------- | -------- |
| write()                 | Excel出力  |
| _add_onchu()            | 「御中」付与   |
| _remove_yen()           | 円削除      |
| _remove_number()        | 成果物番号削除  |
| _replace_date_in_text() | J30の日付置換 |

---

## 4.5 EstimateData

### 役割

抽出データを保持する。

### 保持項目

| 項目             | 型         |
| -------------- | --------- |
| application_no | str       |
| department     | str       |
| subject        | str       |
| outputs        | list[str] |
| amount         | str       |
| due_date       | str       |
| estimate_no    | str       |
| issue_date     | str       |

# 6. 処理フロー

## 6.1 読み込み処理

```text
PDF選択
      │
      ▼
PDFReader.parse()
      │
      ▼
EstimateData生成
      │
      ▼
LedgerReader.find()
      │
      ▼
EstimateData更新
      │
      ▼

```

---

## 6.2 Excel出力処理

```text
Excel出力ボタン
        │
        ▼
保存先選択
        │
        ▼
ExcelWriter.write()
        │
        ▼
テンプレートExcelを開く
        │
        ▼
「フォーマット」シートをコピー
        │
        ▼
新規ブック作成
        │
        ▼
不要シート削除
        │
        ▼
シート名変更（見積書）
        │
        ▼
セルへ転記
        │
        ▼
保存
        │
        ▼
Excel終了
```

# 12. 補足事項

* Microsoft Excelがインストールされている環境で動作する。
* 「TB_見積書フォーマット.xlsx」は新しいものに更新する。
* テンプレートファイルは変更せず、「フォーマット」シートを新規ブックへコピーして利用する。
* 出力ブックには「見積書」シートのみを残す。
* PDFレイアウトや管理台帳の列構成が変更された場合は、抽出・検索ロジックの修正が必要となる。
