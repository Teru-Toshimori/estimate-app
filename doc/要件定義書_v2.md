# 要件定義書 Ver2.0

**システム名**：見積書作成ツール\
**版数**：Ver2.0\
**対象**：Microsoft Graph・OneDrive対応版（タブ機能追加前）

------------------------------------------------------------------------

# 1. システム概要

## 1.1 目的

業務計画書（PDF）から必要な情報を抽出し、OneDrive／SharePoint上の管理台帳へ自動記入・採番を行い、見積書（Excel/PDF）を自動作成することで、転記作業の効率化と入力ミス防止を実現する。

## 1.2 背景

従来はPDF、管理台帳、見積書を手作業で転記していた。 Ver2.0ではMicrosoft
Graph APIを利用し、OneDrive上の管理台帳を直接更新できるようにした。

# 5. 機能要件

## 5.1 PDF解析

抽出項目

-   申請書No
-   依頼部署
-   件名
-   成果物
-   委託金額
-   納期
-   伝票番号（PDFファイル名）

## 5.2 プレビュー

表示項目

-   申請書No
-   伝票番号
-   見積番号
-   発行日
-   依頼部署
-   件名
-   成果物
-   委託金額
-   納期

## 5.3 管理台帳更新

-   Microsoft Graph認証
-   OneDriveから管理台帳取得
-   依頼部署で対象シート検索
-   C列から業務計画書No検索
-   最終行へ追加
-   No採番
-   見積番号採番
-   発行日設定
-   発行者設定
-   OneDriveへ再アップロード

## 5.4 Excel出力

  項目           セル
  -------------- ---------
  見積番号       L1
  操作日         K3
  依頼部署       A7
  件名           E14
  成果物         B18以降
  委託金額       J18
  伝票NO.        B25
  伝票番号       C25
  申請書No       C26
  納期           C30
  納期（日付）   J30

Excel書式・罫線・フォント・列幅を保持する。

## 5.5 PDF出力

Excel出力後にPDFを生成する。
